SIGAP adalah aplikasi Sistem Informasi Geografis (SIG) berbasis web yang digunakan untuk memantau, menampilkan, dan memverifikasi titik-titik kejadian bencana secara real-time.
Proyek ini dibuat sebagai tugas mata kuliah **SISTEM INFORMASI GEOGRAFIS (SIG)**.
